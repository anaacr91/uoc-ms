Arroz meloso de temporada

Una propuesta mediterranea de coccion lenta, pensada para compartir y destacar ingredientes de temporada.

El arroz meloso es una receta versatil que permite adaptar el resultado segun el producto disponible. Para este versionado se prioriza una base de sofrito suave, caldo de verduras casero y setas salteadas para aportar profundidad.

La clave del punto meloso es controlar la hidratacion en tres tandas, removiendo de forma constante solo al final para liberar almidon sin romper el grano. Es recomendable usar arroz de variedad redonda para mantener estructura y textura.

Al emplatar, se finaliza con aceite de oliva virgen extra y perejil fresco picado. Esta terminacion da brillo, aroma y equilibrio al conjunto, transformando un plato humilde en una receta de autor accesible.
