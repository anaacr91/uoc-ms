Tarta rustica de frutas

Postre casero de inspiracion tradicional, con masa quebrada y fruta de temporada para una merienda equilibrada.

La tarta rustica permite una preparacion sencilla y muy visual. La masa se estira de forma irregular para conservar un acabado artesanal, sin necesidad de molde ni tecnicas avanzadas de pasteleria.

Para el relleno se combinan manzana, pera y un toque citrico con canela. La fruta se dispone en capas finas para conseguir coccion uniforme y evitar exceso de humedad en la base.

Antes del horneado se pincela el borde con huevo para obtener color dorado, y al salir del horno se aplica una fina capa de miel. El contraste entre corteza crujiente y centro jugoso aporta gran valor gastronomico.
